## FAQ

**1) How do I know the VPN is actually on?**  
Check the app status shows **Connected**, then confirm your public IP address changed. If your IP didn’t change, you’re not tunneled.

**2) Should I keep Kill Switch enabled all the time?**  
Yes. It prevents traffic leaks when the connection drops. If something breaks (local devices, printers), whitelist local network access if the app supports it.

**3) Which server should I pick for speed?**  
Start with the closest location. If a site is geo-blocked, switch to a server in the required country. Avoid hopping locations unless you need it.

**4) Will this slow down my internet?**  
Some slowdown is normal due to encryption and routing, but you should still get a stable connection. If it’s crawling, change servers or try a different protocol in settings.
